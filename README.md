# E-commerce Sales Dashboard (Power BI)

## 📊 Project Overview

This project is an interactive dashboard built using Power BI to analyze e-commerce sales data. It provides insights into sales, profit, and customer trends.

## 🔧 Tools Used

* Power BI Desktop
* CSV Dataset (Superstore)

## 📈 Features

* Sales and Profit Analysis
* Category-wise Performance
* Region-wise Insights
* Interactive Filters (Slicers)
